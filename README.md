## 📖 About

Texter is a clean, SaaS-inspired text utility tool that helps users quickly transform and analyze their text. Built as a portfolio project to sharpen React and Tailwind CSS skills, it delivers a polished experience across both desktop and mobile.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔠 Uppercase | Convert text to all uppercase |
| 🔡 Lowercase | Convert text to all lowercase |
| ✂️ Remove Spaces | Strip out extra whitespace |
| 📋 Copy Text | One-click copy to clipboard |
| 🗑️ Clear Text | Instantly reset the text area |
| 📊 Word Counter | Live word count as you type |
| 🔢 Character Counter | Live character count |
| ⏱️ Reading Time | Estimated reading time calculator |
| 👀 Live Preview | See your transformed text in real time |
| 📱 Responsive | Fully optimized for mobile & desktop |

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/abdullah-alsaba/Texter.git

# 2. Navigate to the project directory
cd Texter

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be running at `http://localhost:5173`.
